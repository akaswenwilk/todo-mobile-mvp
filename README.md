# Todo Mobile MVP (iOS + Android)

Cross-platform Flutter implementation for a minimalist, dark-first Todo app focused on busy entrepreneurs.

## Scope (MVP)
- Task CRUD
- Reordering tasks
- Completion is separate from deletion
- Single list view
- Local-only data
