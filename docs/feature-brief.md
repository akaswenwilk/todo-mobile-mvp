# Feature Brief — Todo Mobile MVP

## Product
**Working name:** Focus List  
**Platforms:** iOS + Android

## Target User
Busy entrepreneurs managing both work and personal tasks.

## Core Outcome
Reduce cognitive load while ensuring tasks get accomplished.

## Success Metric
Users actively use the app daily for at least 7 consecutive days (D1–D7).

## Product Principles
1. One list, one source of truth
2. Fast capture (<10 seconds to create a task)
3. Completion is **not** deletion
4. Order in list = priority
5. Local-first, on-device only
6. Dark-mode-first visual design

## MVP Scope (Strict)
- Create task
- Read task list
- Update task text
- Delete task
- Mark complete (one-way in MVP)
- Reorder tasks via drag-and-drop
- Persist data locally on device

## Explicit Non-Goals (v1)
- Onboarding
- Dates/reminders/scheduling
- Multiple lists/projects
- Collaboration
- Cloud sync
- Tags, notes, attachments
